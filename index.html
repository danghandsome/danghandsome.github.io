About

Engineer on a mission-critical system.

I'm a C# desktop developer maintaining and extending a large-scale Hospital Information System built on WinForms, DevExpress and a MEF plugin architecture. Over the past year I've shipped features across pharmacy, warehouse, billing, clinical-service and blood-transfusion modules — from UI down to backend API integration and printed medical reports. I trace issues end-to-end across UI, cache, API and data layers, and I use AI coding assistants and prompt engineering to review, refactor and document code faster on a large legacy codebase.
